# Temp assets manifest — sourced gap-filler

Tracking required by `docs/art-pipeline.md` §8: sourced gap-filler looks
finished, so every sourced asset gets a row here when it enters the project
and loses it when final art replaces it. **Pre-ship gate: this table must be
empty.**

Only Kenney (CC0) assets may appear here without explicit written approval
(§7). Block placeholders are tracked separately in `stand_ins.json`,
generated by `tools/make_placeholders.py`.

AssetRegistry parses rows whose first cell is a `res://` path and lists any
still present at startup in debug builds.
